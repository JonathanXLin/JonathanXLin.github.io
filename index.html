General Motors Canada

At General Motors Canada, I developed and implemented an Arduino based handheld continuity tester for seven V8 assembly line operator stations. This project replaced the original plan to purchase a duplicate test unit at every station, resulting in an estimated cost savings of $101,000.

Circuit design and PCB layouts were done using Altium CircuitMaker and Atmel microcontrollers were programmed using C and ICSP.

Department of National Defence

At the Department of National Defence, I developed a genetic algorithm in C and a user interface in C# to optimize operator layouts of high density military workspaces.

I co-authored two published papers on this project's results: DRDC Reference Document (DRDC-RDDC-2018-D139) and an HFES Conference Paper. Both are unclassified and are provided below.

Project Summary

Workspaces are modelled as discrete m x n grids, and operator teams are modelled as directional weighted connections. Each grid element is selected as "open", "barrier", or "inaccessible". Operator desks are represented as sizeable rectangles on the workspace grid.

Starting with a population of initial guesses, successive generations are mutated and evaluated using a fitness function. This fitness function evaluates each candidate solution according to the shortest path distance between each operator pair and its weight. Probabilities of mutation for the next generation are proportional to the fitness of that candidate. After a user specified number of generations, the highest fitness candidate is presented.
