Work Term Evaluation

Official GM end of term evaluation.



At General Motors Canada, I developed and implemented an Arduino based handheld continuity tester for seven V8 assembly line operator stations. This project replaced the original plan to purchase a duplicate test unit at every station, resulting in an estimated cost savings of $101,000.

These handheld test units check engine electrical components for correct continuity, resistance, and/or voltage. Its common design is configurable for use at any continuity test station.

Circuit design and PCB layouts were done using Altium CircuitMaker and Atmel microcontrollers were programmed using C and ICSP.

All project milestones were successfully met, and all continuity test station bypass procedures were updated. Line operators and team leaders were trained to use the handheld testers.
